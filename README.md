# 🐍 Curso de Python: De Básico a Avanzado

¡Bienvenido al repositorio de mi curso de Python! Este proyecto contiene los apuntes, ejercicios y proyectos prácticos que he ido desarrollando a lo largo de mi aprendizaje del lenguaje Python.

## 📁 Estructura del Repositorio

El repositorio está organizado en dos partes principales: **Conceptos Básicos** y **Proyectos Prácticos**.

### 📚 Conceptos Básicos (`/conceptos_basicos`)
Aquí encontrarás la teoría y los ejercicios fundamentales de Python divididos por módulos:

1. **`01_introduccion`**: Conceptos iniciales, variables y sintaxis básica.
2. **`02_listas`**: Manejo de listas, métodos e iteración.
3. **`03_tuplas`**: Uso de tuplas y sus diferencias con las listas.
4. **`04_strings`**: Manipulación de cadenas de texto.
5. **`05_diccionarios`**: Creación y manipulación de estructuras clave-valor.
6. **`06_ciclos_y_condiciones`**: Control de flujo (`if`, `elif`, `else`, `for`, `while`).
7. **`07_funciones`**: Definición de funciones, parámetros y retornos.
8. **`08_clases`**: Programación Orientada a Objetos (POO), herencia, polimorfismo, atributos y métodos.

### 🚀 Proyectos Prácticos (`/proyecto_analisis_datos`)
Contiene aplicaciones prácticas para poner a prueba los conocimientos adquiridos.
- **Customers and Orders Analysis**: Un Jupyter Notebook (`customers_and_orders.ipynb`) donde se analizan datos simulando un entorno de ventas.

## 🛠️ Tecnologías y Herramientas

- **Lenguaje:** Python 3.x
- **Entorno interactivo:** Jupyter Notebooks
- **IDE Recomendado:** VS Code o PyCharm

## 🚀 Cómo usar este repositorio

Si deseas clonar y probar el código en tu máquina local, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/fpaim/Curso-python-.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd Curso-python-
   ```
3. Explora los diferentes módulos dentro de la carpeta `conceptos_basicos` y ejecuta los scripts individualmente:
   ```bash
   python conceptos_basicos/01_introduccion/tipos_datos.py
   ```
